## 💻 Pré-requisitos

Antes de começar, verifique se você atendeu aos seguintes requisitos:
* Você instalou a versão mais recente de `docker compose`

## 🚀 Instalando Organizador de Tarefas

Para instalar o Organizador de Tarefas, siga estas etapas:


```
git clone git@github.com:infingardi/organizador-de-tarefas.git
```
## ☕ Usando Organizador de Tarefas

Para usar Organizador de Tarefas, siga estas etapas:

```
// Para iniciar o projeto
docker compose up --build

// Para Rodar as queries do banco de dados
cd ./backend
npm run db
```
